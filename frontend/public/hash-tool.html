🔐 Black2 文件哈希计算工具

根据 Black2 协议白皮书 2.7 节，计算文件的 SHA-256 哈希值

📁

点击选择文件或拖拽文件到此处

支持任意大小文件（本地计算，不上传）

✅ 哈希计算成功

📖 使用说明

  1. 选择需要发布的商品文件
  2. 点击“计算 SHA-256 哈希”按钮
  3. 等待计算完成（大文件可能需要一些时间）
  4. 复制生成的哈希值
  5. 回到 Black2 平台，在发布商品时粘贴此哈希值
💡 工具选择建议：
• 小文件 (< 500MB)：使用本在线工具
• 大文件 (500MB - 2GB)：下载桌面工具（更快、更稳定）
• 超大文件 (> 2GB)：必须使用桌面工具或 Python CLI
⚠️ 注意：此工具在本地浏览器中计算哈希，文件不会上传到任何服务器，确保您的文件隐私安全。
